# operator-penugasan-java
Program demonstrasi penggunaan operator penugasan (`*=`) dalam bahasa Java.

---

## Fitur Utama
- Demonstrasi operator penugasan perkalian (`*=`)
- Nilai variabel yang mudah dimodifikasi
- Output yang jelas ke konsol

---

## Contoh Output
```
Hasil perkalian: 392
```
